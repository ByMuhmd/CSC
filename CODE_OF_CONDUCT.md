# Code of Conduct

## Our Commitment

We are committed to making participation in this project a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Expected Behavior

- Be respectful and constructive in all discussions.
- Assume good intent and ask clarifying questions before escalating.
- Accept feedback gracefully and focus on improving outcomes.
- Help keep the community welcoming for new contributors.

## Unacceptable Behavior

- Harassment, discrimination, hate speech, or personal attacks.
- Trolling, intimidation, or deliberately disruptive behavior.
- Publishing private information without explicit permission.
- Any conduct that is inappropriate in a professional and educational community.

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this Code of Conduct. Maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that are not aligned with this Code of Conduct.

## Reporting

To report unacceptable behavior, open a private security report using the guidance in [SECURITY.md](SECURITY.md). Include as much context as possible so maintainers can review and act quickly.

## Enforcement

Maintainers may take any action they deem appropriate, including temporary or permanent bans from project spaces, based on the severity and pattern of behavior.

## Scope

This Code of Conduct applies within all project spaces, including issues, pull requests, discussions, and other community interactions.
